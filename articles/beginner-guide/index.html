初学者向け2級ボイラー技士試験の始め方

2級ボイラー技士試験（乙種第4類）を初学者が独学で始める場合、いきなり問題集ではなく「受験資格→試験形式→学習計画」の順が定番です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週に要項で資格確認·40問·240点·各40点足切りをメモし、週4時間×18週＝72時間から始めます。正本はボイラー検定（公式）です。

この記事の要点

本記事を読むと、たとえば「6月資格→7月基礎→9月40問演習→10月本番」の初学者6ステップが分かります。

  • 資格確認が先
  • 40問4科目
  • 80時間目安
  • 演習で測る
  • 要項が正本

1初学者6ステップ（最初の2週間）

初学者は、次の6ステップで開始します。

| # | ステップ |
| --- | --- |
| 1 | 要項で資格·40問·240点確認 |
| 2 | 4科目（構造·取扱い·燃料·法令）把握 |
| 3 | 公式テキスト目次を1周 |
| 4 | 用語解説10語/日 |
| 5 | 演習20問（科目別5問） |
| 6 | 正答率·弱点科目を記録 |

  • たとえば6月6日開始·10月11日13:30試験·残り18週なら
  • 6月第1〜2週で上記6ステップ
  • 6月第3週から週4時間の本格学習
  • と段階を踏み

2試験形式の初学者メモ（40問·400点）

学習前に、試験形式を1枚にまとめます。

| 項目 | 内容 |
| --- | --- |
| 問題数 | 40問（4科目×10問） |
| 満点 | 400点 |
| 合格 | 240点以上 |
| 足切り | 各科目40点以上 |
| 時間 | 13:30〜16:30（約3時間） |

例えば6月第2週の20問演習で28問正解（280点）·法令38点なら不合格、という2級（40問·各40点足切り）の典型例です。「50問·100問」は他資格の誤情報です。

3最初の20問演習で難易度を測る

初学者は、20問演習の正答率で学習時間を見積もります。

| 正答率 | 目安時間 |
| --- | --- |
| 70％+ | 週4h×12週 |
| 50〜69％ | 週5h×18週 |
| 50％未満 | 週6h×18週+ |
| 科目40点- | その科目+3h/週 |

  • たとえば最初の20問で60％（12問正解）なら
  • 80時間前後·週5時間×18週を目標に
  • 4科目のうち最低得点科目を特定してください

合格率より自分の数字を優先します。

4初学者がつまずく4科目の印象

初学者からよく聞く体感です。個人差があるため演習で確認してください。

| 科目 | 初学者の印象 |
| --- | --- |
| 構造 | 用語·装置が多い |
| 取扱い | 現場経験者は有利 |
| 燃料燃焼 | 計算が苦手 |
| 関係法令 | 条文·数値 |

  • 例えば20問演習で構造45点·法令35点なら
  • 残り18週で法令に週3時間·演習20問
  • 構造は週1時間維持
  • という配分が定番

58月申込·10月本番までの初学者ロードマップ

10月11日（日）13:30本番までの例です。

1. 6月：資格·基礎·20問演習
2. 7月：4科目·週4時間
3. 8月：8,800円申込·演習40問
4. 9月：13:30模試·弱点補強
5. 10月：直前1週間·12:45到着

たとえば8月7日まで8,800円決済·9月第1週日曜40問模試·10月10日20:00持ち物、をカレンダーに書いてから学習を続けます。

6よくある質問

初学者は何から手を付ければよいですか？
受験資格の確認が最優先です。たとえば6月第1週に要項表·講習修了証·実務年数を照合し、資格確定後に40問·240点·各40点をメモ、6月第2週から公式テキスト目次·演習20問、の順が定番です。資格なしで8,800円申込は返金なしのおそれがあります。
初学者向けの学習時間の目安は？
80時間前後·残り18週なら週4〜5時間です。例えば最初の20問演習60％なら週5時間×18＝90時間を目標に、2週間ごとに同じ20問を解き直し、70％以上になるまで弱点科目に時間を追加してください。初学者向け難易度記事も合わせて読んでください。
指定講習前からテキストは読めますか？
資格·講習修了前でも、試験形式·用語の予習は可能です。たとえば講習受講中·残り18週なら、構造·取扱いの用語10語/週と演習10問/週から始め、講習修了·資格確定後に8,800円申込、という流れが安全です。申込前に資格確認記事で書類を照合してください。
