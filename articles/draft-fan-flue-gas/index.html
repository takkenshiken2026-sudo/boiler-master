吸送風機と排ガス｜2級ボイラー技士の構造・燃焼対策

吸送風機（ドラフトファン）と排ガスは、2級ボイラー技士試験で構造科目（各10問）と燃料燃焼科目（各10問）の両方から問われる横断論点です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、「送風·排気·空気比·排ガス温度」の4列比較表を作り、構造·燃料各5問ずつ解く、という2科目セット学習から始めるのが定番です。

この記事の要点

本記事を読むと、たとえば吸送風機の役割と排ガス·空気比の関係が表で整理され、構造·燃料10問ずつの解き方が分かります。

  • 構造·燃料横断
  • 4列比較表
  • 空気比連動
  • 各10問
  • 40点足切り

1吸送風機の役割（構造·燃料の接点）

吸送風機は、燃焼用空気と排ガス排出の両方に関わります。

| 観点 | 科目 |
| --- | --- |
| 装置構造 | 構造10問 |
| 空気·排ガス | 燃料10問 |
| 異常時 | 両科目 |
| 正本 | 公式テキスト |

  • たとえば6月第1週4列比較表を写し
  • 10月11日13:30本番まで残り18週·構造5問+燃料5問/週
  • と2科目セットで回すのが定番

50問試験等の他資格情報と混同しないでください。

2排ガス温度·成分の試験ポイント

排ガスは、燃焼状態の指標として問われます。

| 項目 | 試験の焦点 |
| --- | --- |
| 排ガス温度 | 空気比·効率 |
| 成分 | CO·過剰酸素等 |
| 異常 | 過多·不足 |
| 誤答 | 原因·結果逆 |

  • 例えば6月第2週燃料5問で排ガス温度2問·空気比1問
  • 構造5問で吸送風機2問
  • 誤答3問を比較表に追記
  • 7月第1週10問解き直し
  • を目標にしてください

| 週 | 内容 |

4空気比·燃焼不良との連動復習

吸送風機·排ガスは、燃焼不良の原因表とセットです。

| 連動 | 内容 |
| --- | --- |
| 空気不足 | 黒煙·不完全燃焼 |
| 空気過多 | 排ガス温度↑ |
| 風量不足 | 排ガス滞留 |
| 対策 | ダンパ·ファン |

  • 例えば6月燃焼不良3原因表·6月第3週本記事10問·7月燃焼分野過去問10問
  • 8月8
  • 800円申込前に燃料44点以上
  • という順が定番

59月模試までの10問×8セット計画

残り18週の具体例です。

1. 6〜7月：構造5+燃料5×4週
2. 8月：弱点10問集中·8,800円申込
3. 9/14：13:30·40問模試
4. 9/21：解き直し20問

  • 燃料44点なら燃料維持·法令·取扱いに時間シフト
  • という調整が定番

6よくある質問

吸送風機は構造だけで学習しますか？
構造·燃料の両方から出ます。たとえば6月比較表·構造5問+燃料5問/週、10月11日13:30本番まで計40問（構造20+燃料20）、解き直し20問、が定番の量です。構造科目40点·燃料40点、両方の足切りを意識してください。
排ガス温度は計算問題ですか？
理論選択·数値読み取り·空気比計算のいずれかです。例えば燃料10問中1問が排ガス温度·1問が空気比計算なら、計算問題の解き方記事と6月第2週1時間セット、9月燃料42点以上、を目標にしてください。数値基準は公式テキストで確認します。
燃焼不良記事との使い分けは？
本記事は吸送風機·排ガスの装置·空気流れが中心、燃焼不良記事は黒煙·不完全燃焼の原因·対策、燃焼分野過去問記事は燃料10問の解き直しサイクル、と役割分担してください。たとえば6月本記事·7月不良·8月過去問、9月40問模試各40点以上、が定番の順序です。
