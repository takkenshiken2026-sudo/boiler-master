2級ボイラー技士試験の再受験の日程調整

2級ボイラー技士試験の再受験では、次回試験日·申込締切·学習期間をセットで調整する必要があります。たとえば10月11日（日）13:30回不合格·11月合格発表後なら、翌春回の申込開始（例2月）から逆算し、40点未満科目に残り18週×3時間を配分するのが定番です。再受験も8,800円·40問·要項どおりの申込が必要です。

この記事の要点

本記事を読むと、たとえば「11月不合格→翌春申込→18週で法令20問」の日程·学習の組み立て方が表で整理されます。

  • 次回日程逆算
  • 8,800円再申込
  • 18週学習
  • 足切り40点優先
  • 繁忙期回避

1不合格発表後の日程確認手順

合格発表後、次の3点を公式日程表で確認します。

| 項目 | 確認 |
| --- | --- |
| 次回試験日 | 13:30開始·40問 |
| 申込開始·締切 | 8,800円決済期限 |
| 合格発表 | その次の再受験候補 |

たとえば10月11日回·11月15日合格発表·不合格なら、翌春（例4月13:30回）の申込2月開始をカレンダーに転記します。ボイラー検定（公式）の年間表が正本です。

2春期·秋期の選び方（再受験者向け）

再受験者は、学習期間と仕事繁忙期のバランスで受験回を選びます。

| 回 | メリット | 注意 |
| --- | --- | --- |
| 早め（翌春） | 記憶鮮度 | 申込2月等 |
| 遅め（翌秋） | 学習余裕18週+ | モチベ維持 |
| 繁忙期回避 | 学習時間確保 | 日程表確認 |
| 併願 | 別日なら可 | 同日13:30不可 |

  • 例えば11月不合格·現場繁忙期1〜3月なら
  • 翌秋10月11日13:30回を候補に
  • 4〜9月に週4時間×24週＝96時間計画が現実的

340点未満科目別の逆算スケジュール

前回の科目別得点から、次回までの週次計画を組みます。

| 弱点 | 18週目安 |
| --- | --- |
| 法令38点 | 週3時間·演習20問 |
| 燃料40点- | 週2時間·計算10問 |
| 総合230点 | 4科目バランス |
| 全科目40点+ | 模試·維持 |

  • たとえば関係法令38点·残り18週なら
  • 6月第1週から法令用語10語·演習20問
  • 9月に40問模試·10月11日13:30本番
  • という流れをカレンダーに書き

4再申込タイミング（8,800円）

申込は次回の申込開始日から、締切3日前までに8,800円決済を完了させます。

| タイミング | 行動 |
| --- | --- |
| 申込2か月前 | 受験資格再確認 |
| 申込開始日 | 受験地·会場選択 |
| 7日以内 | 8,800円決済 |
| 受験票到着 | 13:30·会場照合 |

  • 例えば2月1日申込開始·4月13:30試験なら
  • 2月7日までに決済·3月に40問模試1回·4月直前1週間
  • という流れが定番

5再受験と仕事·講習の調整

指定講習修了済みなら資格は維持されますが、実務年数の起算·申込日時点の要件は要項で再確認します。

1. 受験資格表を申込前に再照合
2. 残業期と学習週3時間の確保
3. 40問·240点·各40点の目標設定
4. 本番7日前模試·前日20:00持ち物
5. 12:45到着·13:30開始

  • たとえば再受験2回目·前回280点·法令38点なら
  • 次回まで法令に70％の学習時間を配分し
  • 他科目は間違いノート5問/日で維持し

6よくある質問

不合格後、すぐ次の回に申し込むべきですか？
学習期間·繁忙期·40点未満科目の対策時間を見て選びます。たとえば10月11日不合格·弱点法令で演習20問に6週必要なら、翌春回（18週以上先）の方が足切り40点対策が現実的です。早めの回を選ぶ場合も、8,800円決済前に週次計画を立ててください。
再受験の日程変更はできますか？
申込後の試験日変更は、要項上原則できないことが多いです。例えば4月13:30回を申し込んだ後、10月回へ変更したい場合、期限内の変更手続がなければ取消·次回再申込（8,800円）が必要です。申込前に日程表で繁忙期と照合してください。
2回連続不合格の場合の調整は？
2回分の40問得点記録を並べ、毎回40点未満の科目を特定します。たとえば1回目法令38点·2回目燃料39点なら、3回目は両科目に週2時間ずつ·残り18週で演習各20問、と優先順位を付け直します。再受験のたび8,800円·40問·13:30開始は同じです。再受験戦略記事も合わせて参照してください。
