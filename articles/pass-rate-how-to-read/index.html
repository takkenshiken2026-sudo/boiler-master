2級ボイラー技士試験の合格率・統計の読み方

2級ボイラー技士試験の合格率·統計は、「何人中何％が合格したか」を示しますが、個人の合格見込みとは一致しません。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、公式発表50％前後の年でも40問演習320点·全科目40点以上なら本番合格圏内、演習200点なら厳しい、と読み替えます。正本はボイラー検定（公式）の公表資料です。

この記事の要点

本記事を読むと、たとえば「公式50％·演習260点·法令38点」の3列メモで合格率を読み、口コミの数字と公式統計を混同しなくなります。

  • 公式統計が正本
  • 3列で比較
  • 足切りを忘れない
  • 演習得点優先
  • 古い数字注意

1合格率の定義（何％の意味か）

合格率＝合格人数÷受験人数×100（おおむね）です。定義の細部は公式発表に従います。

| 用語 | 意味 |
| --- | --- |
| 受験者数 | その回の受験人数 |
| 合格者数 | 合格基準を満たした人数 |
| 合格率 | 合格者÷受験者 |
| 足切り | 各科目40点以上も必要 |

たとえば受験者1,000人·合格者500人なら合格率50％前後ですが、あなた個人の40問演習280点·法令38点ならその統計とは無関係に不合格リスクがあります。

23列メモ表の作り方

公式統計と自分の演習を並べると、誤読が減ります。

| 列 | 記入例 |
| --- | --- |
| A：公式合格率 | 50％（2024秋期·要確認） |
| B：40問演習得点 | 260点（6月第1週） |
| C：40点未満科目 | 関係法令38点 |

例えばA列50％·B列260点·C列法令38点なら、合格率は高めでも法令対策が最優先です。2週間後にB列を同じ40問セットで更新し、280点·全科目40点以上を目指してください。

3よくある誤読5つ

合格率統計を読む際の典型ミスを整理します。

| 誤読 | 正しい読み方 |
| --- | --- |
| 50％＝半分は勉強不要 | 個人の演習得点で判断 |
| 1級の合格率混同 | 2級40問の公式値 |
| 50問試験の数字 | 2級は40問 |
| 古い年度の数字 | 最新公表値 |
| 総合点だけ見る | 各科目40点必須 |

たとえば「2級50問·合格率80％」というブログ記事は2級（40問）と無関係のおそれがあります。ボイラー検定（公式）の2級学科試験の数字だけを参照してください。

4受験者数と合格率のセット読み

受験者数が少ない回は、合格率が大きく振れることがあります。

| パターン | 読み方 |
| --- | --- |
| 受験者多·合格率中 | 参考程度 |
| 受験者少·合格率極端 | 慎重に |
| 複数回平均 | 推移を見る |
| 自分の演習 | 最優先指標 |

例えば受験者200人·合格率65％と受験者2,000人·合格率45％では母数が異なり単純比較できません。10月11日13:30本番に向け、自分の40問得点の伸びを2週間ごとに記録してください。

5統計を読んだあとの行動

合格率を確認したら、学習計画に反映します。

1. 240点·各40点の合格基準を要項で再確認
2. 40問演習1回·4科目得点を記録
3. 40点未満科目に週3時間追加
4. 80時間前後の総学習時間を逆算
5. 2週間ごとに同じ40問で解き直し

たとえば6月6日·10月11日試験まで18週·演習220点なら、合格率50％前後に関わらず週5時間×18＝90時間計画が現実的です。

6よくある質問

合格率と合格点の違いは何ですか？
合格率は「受験者のうち合格した割合（統計）」、合格点は「240点以上·各科目40点以上（個人の基準）」です。たとえば合格率55％の年でも、40問演習200点なら個人としては合格基準未達です。合格点記事で240点·400点満点·4科目×各10問の構造も確認してください。
複数年度の合格率を平均してよいですか？
推移を見る参考にはなりますが、学習量の決定には自分の演習得点を優先してください。例えば直近3年平均50％前後でも、6月の40問演習240点·法令38点なら法令演習20問を残り18週で追加する方が合格に直結します。公式の最新年度·最新回の数字を必ず確認してください。
統計を見て不安になったらどうすればよいですか？
40問演習を1回解き、320点以上·全科目40点以上かを確認してください。たとえば演習300点·全科目42点以上なら、合格率40％前後の年でも合格見込みは高めです。逆に240点未満なら、残り18週で80時間前後の学習計画を立て直す方が統計を読むより有効です。難易度記事も合わせて参照してください。
