2級ボイラー技士試験の免除制度

2級ボイラー技士試験（乙種第4類）に免除·短縮規定があるか、自分が該当するかは、口コミではなくボイラー検定（公式）の最新受験要項が正本です。たとえば6月第1週に要項PDFで「免除」を検索し、該当条項があれば要件表で1行を特定、なければ通常の受験資格表から10月11日（日）13:30試験まで残り18週の学習計画に進むのが定番です。正本は一般社団法人 日本ボイラー協会の最新要項です。

この記事の要点

本記事を読むと、たとえば要項検索→該当行特定→書類準備→8,800円決済前確認の手順が分かり、非公式情報だけで40問試験を誤解しません。

  • 要項PDF検索
  • 該当1行特定
  • 関連資格·短縮
  • 該当なしは通常区分
  • 8,800円前に確認

1要項PDFで「免除」を検索する手順

免除·短縮規定の有無は、最新要項PDF内検索から始めます。

| 手順 | 内容 |
| --- | --- |
| 1 | ボイラー検定（公式）で最新要項PDF取得 |
| 2 | 「免除」「短縮」「特例」で検索 |
| 3 | 該当章があれば要件表を印刷 |
| 4 | 該当1行を特定 |
| 5 | 必要証明書リストを作成 |

  • たとえば6月6日に要項PDFで「免除」を検索し
  • 該当0件なら通常の受験資格表（講習·実務·学歴）へ
  • 該当ありなら要件表の1行をメモし

体験記の「全部免除」は採用しないでください。

2関連資格保有と実務年数短縮

要項によっては、関連資格保有で実務経験年数が短縮される区分がある場合があります。個人の該当は必ず要項正本で確認してください。

| 確認軸 | 例 |
| --- | --- |
| 保有資格 | 要項記載の資格名 |
| 実務短縮 | 3年→2年等（要項確認） |
| 講習 | 指定講習修了は別途必要な場合 |
| 証明 | 資格証·経歴書 |

例えばボイラー取扱作業主任者等を保有していても、2級ボイラー技士試験（40問·4科目）そのものが自動免除されるわけではありません。要項表の該当行で短縮·免除の範囲を確認してください。

3学科試験40問と免除の関係

2級ボイラー学科試験は40問·400点·4科目×各10問です。科目免除·問題数短縮があるかは要項次第です。

| 項目 | 2級の基本 |
| --- | --- |
| 問題数 | 40問（各10問） |
| 合格 | 240点·各40点以上 |
| 時間 | 13:30〜16:30 |
| 免除 | 要項の特例章で確認 |

たとえば「50問→35問」等の口コミは他資格·誤情報の可能性があります。2級は常に40問·各科目40点足切りを前提に、要項に特例があればその範囲だけ適用してください。

4該当なしの場合の通常受験フロー

要項検索で免除条項がない·該当しない場合は、通常の受験資格表から申込みます。

| ステップ | 内容 |
| --- | --- |
| 1 | 講習修了·実務·学歴を要項表で確認 |
| 2 | 8月申込開始·8,800円決済 |
| 3 | 40問·4科目の学習計画 |
| 4 | 10月11日13:30本番 |
| 5 | 合格240点·各40点以上 |

  • 例えば6月6日時点で免除該当なし·講習修了·実務3年なら
  • 通常区分で8月申込·残り18週で4科目学習
  • 9月に40問演習1回を実施する流れが定番

5申込前の問い合わせと書類準備

該当可否が曖昧な場合は、8,800円決済前に日本ボイラー協会へ問い合わせます。

1. 保有資格証·講習修了証·経歴書の写しを用意
2. 希望試験日（例10月11日13:30）を伝える
3. 要項の該当行·条項番号を確認
4. 回答をメモ·PDF保存
5. 資格確定後に8,800円決済

たとえば6月第1週問い合わせ·6月第2週資格確定·8月申込なら、残り16週を40問対策に充てられます。口コミだけで8,800円を支払わないでください。

6よくある質問

免除制度はどこで確認しますか？
ボイラー検定（公式）の最新受験要項PDFが正本です。たとえば6月第1週に要項PDFで「免除」を検索し、該当章があれば要件表で1行を特定してください。該当がなければ通常の受験資格表から10月11日（日）13:30試験までの学習計画に進みます。年度で条項が変わるため、受験予定年度の要項だけを参照してください。
関連資格があれば試験が免除されますか？
関連資格で実務年数短縮等がある場合は要項表に記載があります。例えば取扱作業主任者保有でも、2級40問学科試験全体が自動免除されるとは限りません。指定講習修了·実務年数·学歴の組み合わせは要項正本で確認し、不明点は日本ボイラー協会へ問い合わせてから申込してください。
免除該当の確認はいつまでにすべきですか？
8,800円決済·申込締切の前までに完了させてください。たとえば8月1日申込開始·8月下旬締切なら、6月第1週に要項検索·7月に書類準備·8月7日までに該当可否を確定する流れが安全です。確認が遅れると、資格不備のまま決済し返金なしとなるおそれがあります。受験資格記事と併せて要項表を照合してください。
