2級ボイラー技士試験のボイラーの取扱いに関する知識の計算・数値問題

2級ボイラー技士試験の取扱い科目（各10問）では、水位·水質（pH·硬度·ブローダウン率）·蒸気量などの数値読み取り·計算が1〜3問程度出ます。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週に取扱い演習10問のうち数値3問を単位チェック付きで解き、水質論点の比較表を1枚作る、から始めるのが定番です。

この記事の要点

本記事を読むと、たとえば取扱い10問中1〜3問の数値·計算問題とpH·硬度の解き方が分かります。

  • 数値1〜3問
  • 水質比較表
  • 取扱い10問
  • 40点足切り
  • 週2時間

1取扱い10問中の数値·計算配分

取扱い科目10問中、数値はおおむね1〜3問です。

| タイプ | 例 |
| --- | --- |
| pH·硬度 | 管理基準 |
| ブローダウン率 | 3％等 |
| 水位·蒸気 | 読み取り |
| 理論のみ | 7〜9問 |

  • たとえば6月第1週取扱い10問·数値2問中1問不正解なら
  • 水質論点に週2時間を集中
  • 10月11日13:30本番まで残り18週
  • と配分を決め

2水質数値の比較表（pH·硬度·ブロー）

水質論点は、1枚表で定着させます。

| 項目 | 試験の焦点 |
| --- | --- |
| pH | 管理範囲 |
| 硬度 | 給水5mg/L以下等 |
| ブローダウン率 | 3％未満等 |
| 正本 | 公式テキスト |

  • 例えば6月第2週比較表写し10分·取扱い数値5問·解説20分
  • 火·金に1時間固定·8月8
  • 800円申込前まで12回
  • 脱気器記事と往復してください

33ステップ解き方（数値問題）

数値問題は手順固定でミスを減らします。

| ステップ | 内容 |
| --- | --- |
| 1 | 求める量·単位を下線 |
| 2 | 基準値を表から確認 |
| 3 | 大小·範囲で選択 |
| 4 | 1問4分で次へ |

  • たとえば9月13:30·40問模試で取扱い38点·数値2問不正解なら
  • 10月4日まで数値10問·10月11日12:45到着·本番
  • という直前対策が定番

280点·取扱い38点は不合格です。

4誤答3分類（単位·読み取り·混同）

10問の誤答を分類すると対策が速くなります。

| 分類 | 対策 |
| --- | --- |
| 単位 | 表追加 |
| 読み取り | 下線練習 |
| 混同 | 脱気器·ブロー比較 |
| 桁 | 筆算 |

  • 例えば7月第1週10問中
  • 読み取り3·混同2·単位1なら
  • 7月第2週ブロー管理記事10問
  • 8月第1週再分類で混同0
  • 9月取扱い42点以上
  • を目標にしてください

6よくある質問

取扱いに計算は多く出ますか？
10問中1〜3問程度が目安です。たとえば6月10問·数値2問·理論8問なら週2時間を数値、10月11日13:30本番まで計15問+解き直し5問、9月取扱い40点以上2回、が定番です。数値基準は公式テキストで必ず確認してください。
取扱い基礎記事との違いは？
基礎記事は4ブロック·20問測定、本記事は数値·計算1〜3問の比較表·3ステップ解き方、が中心です。例えば6月基礎·6月第3週本記事、7月脱気器·ブロー記事、9月240点·各40点以上、という順が定番です。
燃料の計算記事と共通ですか？
3ステップ解き方は共通、論点は水質·水位が中心、燃料は空気比·排ガス、です。たとえば6月燃料計算10問·7月取扱い数値10問、9月40問模試で燃料·取扱い各40点以上、を別記録してください。40問·各40点足切りを維持します。
