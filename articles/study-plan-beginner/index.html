2級ボイラー技士試験の初学者向け学習計画

2級ボイラー技士試験（40問·400点·180分）の初学者向け学習計画は、最初の20問演習で正答率を測り、80時間前後·週4時間×18週から逆算するのが定番です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週20問演習·正答率記録·6月第2週から週4時間·8月8,800円申込·9月模試240点·各40点以上2回、という初学者ロードマップから始めます。

1初学者80時間の逆算（20問測定から）

計画は20問演習の正答率から始めます。

| 正答率 | 目安 |
| --- | --- |
| 70％+ | 週4h×18週 |
| 50〜69％ | 週5h×18週 |
| 50％未満 | 週6h×18週+ |
| 40点未満科目 | +3h/週 |

たとえば6月第1週20問12問正解（60％）·10月11日13:30本番まで残り18週·週5時間×18＝90時間、最低得点科目に週3時間追加、9月模試240点·各40点以上2回、10月11日13:30本番まで、という調整が定番です。

6よくある質問

初学者は何時間必要ですか？
80〜90時間が目安です。たとえば6月20問60％·週5時間×18＝90時間·9月模試240点·各40点以上2回、20問測定で週4時間に下げられる場合もあります。10月11日13:30本番まで2週ごと再測定が定番です。
始め方記事との違いは？
本記事は80時間逆算·月別配分、始め方記事は資格確認·6ステップ、が中心です。例えば6月始め方資格確認·6月第2週本記事20問測定、9月40問模試240点·各40点以上2回、10月11日13:30本番まで両方参照が定番です。
指定講習前から計画できますか？
用語·試験形式の予習計画は可能です。たとえば講習受講中·残り18週·週4時間·用語10語/週、資格確定後8,800円申込·9月模試240点·各40点、400点満点·180分が本番条件です。10月11日13:30本番まで週4時間維持が定番です。
