2級ボイラー技士試験｜関係法令の対策｜検査・届出・作業主任者を整理する

2級ボイラー技士試験の関係法令科目（各10問）では、届出·検査·取扱作業主任者·ボイラー区分の4論点がセットで出題されます。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週に4論点比較表を作り、法令演習10問でタグ分類·誤答記録から始めるのが定番です。240点·各科目40点以上を常に意識してください。

この記事の要点

本記事を読むと、たとえば関係法令4論点表と法令10問の週次演習·9月40問模試への接続が分かります。

  • 4論点表
  • 法令10問

1関係法令4論点の比較表（届出·検査·主任者·区分）

法令科目は4論点表で全体像を固定します。

| 論点 | 試験の焦点 |
| --- | --- |
| 届出 | 設置·変更·廃止 |
| 検査 | 性能·再検査·検査証 |
| 主任者 | 選任·職務·監督 |
| 区分 | 小型·伝熱面積 |

  • たとえば6月第1週4論点表を写し
  • 10月11日13:30本番まで残り18週·法令10問中2〜3問が検査·届出
  • と想定して週3時間配分し

数値·期限は公式テキストの規定どおり確認してください。

2法令10問の週次演習サイクル

初回は正答率よりタグ分類が目的です。

| 週 | 内容 |
| --- | --- |
| 6月第1週 | 法令10問通し |
| 6月第2週 | 同10問解き直し |
| 7月 | 弱点論点記事 |
| 9月 | 40問模試 |

  • 例えば6月6日法令10問·7問正解70点·届出3問不正解なら
  • 7月は届出記事·演習10問
  • 9月13:30·40問模試で240点·各40点以上を確認し

3主体の取り違え（事業者·主任者·2級技士）

法令の失点原因は主体·制度の混同です。

| 主体 | 典型の役割 |
| --- | --- |
| 事業者 | 届出·選任 |
| 主任者 | 取扱監督 |
| 2級技士 | 検査関連 |
| 誤答 | 役割入替 |

  • たとえば6月第2週法令10問·主任者2問不正解なら
  • 主任者記事の3列比較表と往復·7月第1週10問解き直し3問正解
  • を目標にしてください

4関連記事とのセット復習

4論点は個別記事とセットで定着します。

| 論点 | 記事 |
| --- | --- |

| 検査 | 検査証·種類 |
| 主任者 | 職務·選任 |
| 過去問 | 法令解き方 |

  • 例えば6月本記事4論点表·7月届出10問·検査10問
  • 8月8
  • 800円申込前まで法令30問·9月法令42点以上2回
  • という流れが定着しやすい

6よくある質問

関係法令は暗記量が多いですか？
4論点表+主体理解が中心です。たとえば6月4論点表·7月弱点記事10問·9月40問模試240点·各40点、条文丸暗記より手続·主体の比較表を優先してください。期限数値は公式テキストで必ず確認します。240点·各40点の基準は不変です。
他3科目との時間配分は？
法令40点未満なら週3時間追加が目安です。例えば6月40問模試220点·法令38点·取扱い45点なら、7月は法令2時間·取扱い1時間、各科目40点足切りを最優先に配分を調整してください。総合240点以上も同時に確認します。
法令過去問記事との違いは？
本記事は4論点の全体像、過去問記事は10問の解き方·3日7日サイクル、が中心です。たとえば6月本記事4論点表、6月第2週から過去問記事の10問サイクル、9月40問模試まで両方参照が定番です。4論点表は月1更新も有効です。
