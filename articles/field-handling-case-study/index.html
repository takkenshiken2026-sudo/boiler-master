1事例問題の4パターン（水位·点火·異常·停止）

事例型は、状況描写から正しい操作を選びます。

| 状況 | 典型判断 |
| --- | --- |
| 水位低下 | 給水·原因確認 |
| 満水 | ブロー·減少 |
| 点火失敗 | 安全確認 |
| 異常停止 | 手順順守 |

  • たとえば6月第1週4事例表を写し
  • 水位·点火の試験論点を整理
  • 10月11日13:30本番まで残り18週·取扱い10問中2〜3問が事例型
  • と想定して週2時間配分し

6よくある質問

現場経験があれば事例は楽ですか？
演習得点で判断してください。たとえば6月10問·事例3問正解なら週1時間維持、2問以下なら週3時間·4事例表復習、9月取扱い40点以上2回、が定番です。試験の正式手順と現場慣行は異なる場合があります。
燃焼安全装置との関連は？
点火·異常停止の事例は燃焼安全装置論点とセットで出得ます。例えば6月本記事4事例·7月燃焼安全10問·解き直し5問、9月40問模試取扱い42点以上、失火·逆火の混同に注意、各40点足切りも確認してください。
取扱い頻出論点記事との違いは？
頻出論点記事は論点リスト、本記事は事例型の状況→判断·4パターン表、が中心です。たとえば6月基礎·7月頻出·8月本記事事例10問·解き直し5問、9月240点·各40点以上2回、10月11日13:30本番まで、という順が定番です。
