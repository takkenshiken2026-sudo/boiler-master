脱気器と給水処理｜2級ボイラー技士の取扱い・水質対策

脱気器と給水処理は、2級ボイラー技士試験の「ボイラーの取扱いに関する知識」（各10問）で頻出の水質論点です。たとえば試験4か月前·残り16週の学習では、脱気器の目的（溶解酸素除去）と給水硬度·pH管理をセットで押さえる必要があります。本記事では現場の水質管理と試験の四択で問われる論点を表に整理します。

この記事の要点

本記事を読むと、たとえば取扱い10問中2問の水質演習で脱気器と軟水装置の目的を区別できるようになり、40点足切りラインの得点損失を防げます。

  • 脱気器＝溶存酸素除去
  • 給水硬度·pH管理
  • 取扱い10問中の水質
  • 連続ブローとセット
  • 規程·基準値は要確認

1脱気器の目的と試験での問われ方

脱気器（デアレータ）。

  • 給水に溶け込んだ酸素·二酸化炭素を除去し
  • ボイラー内部の腐食·ポンピングを防ぐ装置

2級では「何を除去するか」「脱気不足で何が起きるか」が四択で問われます。

| 項目 | 内容 |
| --- | --- |
| 主目的 | 溶存酸素（O₂）の除去 |
| 副次効果 | CO₂除去·給水加温 |
| 脱気不足 | 酸素腐食·ポンピング |
| 試験科目 | 取扱い（各10問） |

  • たとえば取扱い10問中2問が水質関連で
  • 「脱気器は硬度50mg/Lを除去する」という肢が出た場合
  • 溶存酸素除去が目的の脱気器では誤答

硬度除去は軟水装置·イオン交換装置の役割として区別してください。用語解説で「脱気器」「溶存酸素」を確認し、取扱い演習10問で装置目的の問題を優先的に解いてください。

2給水処理の基本（硬度·pH·脱塩）

給水処理は、ボイラーへの障害物質（硬度成分·塩分·酸素等）を許容範囲以下に抑える一連の処理です。試験では処理方法と目的の対応が頻出です。

| 処理 | 目的 | 典型装置 |
| --- | --- | --- |
| 軟化 | 硬度低減 | イオン交換装置 |
| 脱塩 | 塩分除去 | RO·混床 |
| pH調整 | 腐食·スケール抑制 | 薬品注入 |
| 脱気 | 酸素除去 | 脱気器 |

  • 例えば取扱い10問中2問で
  • 給水硬度80mg/L·pH8.3の条件が出題された場合
  • 炉内スケール付着·伝熱低下につながるため
  • 軟水処理の後段で脱気器へ送る流れを図示して覚えると定着しやすい

管理基準値は運転基準·水質管理基準で確認し、数値は規程どおり要項·基準表が正本です。

3脱気器と給水ラインの位置づけ

給水ラインでは、原水からボイラー給水に至るまで複数の処理工程があります。処理順序·装置配置の取り違えが誤答肢になりやすいです。

| 工程（例） | 装置·処理 |
| --- | --- |
| 前処理 | ろ過·軟化 |
| 脱気 | 脱気器 |
| ボイラー入口 | 給水ポンプ |
| 炉内 | 連続ブロー |

たとえば10月11日試験まで残り12週の学習で、「軟化の前に脱気器を置く」など順序が逆の肢が四択に出ることがあります。現場のフロー図を1枚描き、各装置の入口·出口に「除去する物質」をメモしてください。水質管理記事と往復すると理解が深まります。

4連続ブロー·ブローダウンとの関係

給水処理だけでは炉水の濃縮は止まらないため、連続ブロー（ブローダウン）で濃度を管理します。脱気器·給水処理とセットで出題されることが多い論点です。

| 項目 | 関係 |
| --- | --- |
| 給水処理 | 入口の水質確保 |
| 炉水濃縮 | 蒸発による濃度上昇 |
| 連続ブロー | 濃縮抑制 |
| ブローダウン率 | 管理指標 |

例えば給水硬度5mg/L以下でも、ブローダウン率3％未満で炉水が濃縮すればスケール·起沫のリスクが残ります。演習で「給水処理不要·ブローのみで十分」型の誤答肢に注意し、両方必要という正解パターンを体得してください。

5取扱い科目10問での学習の進め方

取扱い科目は各10問·100点満点で、合格には科目別40点以上（足切り）が必要です。水質だけ得意でも他論点で40点割れは不合格になります。

1. 脱気器·給水処理の用語10語を用語解説で確認
2. 取扱い演習10問を1セット解き、水質の正答率を記録
3. 誤答は「装置目的」「処理順序」「管理指標」に分類
4. 連続ブロー·低水位警報と横串で比較表を作成
5. 1週間後に同じ10問を解き直し正答率70％以上を目標

  • たとえば最初の10問演習で取扱い6問正解（60％）なら
  • 水質3問のうち1問が脱気器目的の取り違え
  • という粒度で弱点を特定してください

取扱い科目対策記事と組み合わせ、週2時間を水質に配分するのが定番です。

6よくある質問

脱気器は何を除去しますか？
主に給水中の溶存酸素（O₂）を除去します。二酸化炭素除去·給水加温も兼ねる場合があります。たとえば「硬度を除去する装置」として脱気器を選ぶ肢は誤答です。硬度は軟水装置等の役割として区別してください。取扱い演習で装置と目的の組み合わせ問題を10問解くと定着しやすいです。
給水処理と炉水管理の違いは？
給水処理はボイラー入口前の水質を整える工程、炉水管理は運転中の炉水濃度·pH等を連続ブロー等で維持する工程です。例えば軟水装置で給水硬度5mg/L以下にしても、ブローダウン率3％未満で炉水が濃縮すれば障害は起き得ます。両方必要というのが試験の基本線です。比較表で「入口 vs 炉内」を整理してください。
試験ではどの程度詳しく出ますか？
2級では装置の目的·処理順序·管理指標の理解が中心で、詳細な化学計算より「何を防ぐか」の論理が問われます。たとえば取扱い10問中1〜2問が水質関連、脱気器·pH·硬度·連続ブローのいずれかが出るイメージです。数値基準は運転基準·水質管理基準で確認し、取扱い演習で過去問形式を10問通しで解いてください。
