ボイラー検査証と性能検査｜法令問題で迷いやすい手続

ボイラー検査証と性能検査は、2級ボイラー技士試験の関係法令科目（各10問）で手続·主体·期限の取り違えが頻出の論点です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週に「検査証·性能検査·再検査·届出」の4行比較表を作り、法令演習10問で検査タグの誤答を記録する、から始めるのが定番です。

この記事の要点

本記事を読むと、たとえば検査証·性能検査の手続比較表と法令10問中1〜3問の解き方が分かります。

  • 4行比較表
  • 検査論点
  • 主体混同
  • 法令10問
  • 40点足切り

1検査証·性能検査·再検査·届出の4行比較

法令の検査論点は4行表で定着させます。

| 項目 | 試験の焦点 |
| --- | --- |
| 検査証 | 交付·記載 |
| 性能検査 | 定期·再検査 |
| 再検査 | 不合格時 |
| 届出 | 設置·変更 |

  • たとえば6月第1週4行表を写し
  • 10月11日13:30本番まで残り18週·法令10問中2問が検査系
  • と想定して週2時間配分し

期限数値は公式テキストの規定どおり確認してください。

2主体の取り違え（事業者·2級技士·検査機関）

検査手続の失点原因は主体の混同です。

| 主体 | 典型の役割 |
| --- | --- |
| 事業者 | 届出·選任 |
| 2級技士 | 検査関連 |
| 検査機関 | 性能検査実施 |
| 誤答 | 役割入替 |

  • 例えば6月第2週法令10問·検査3問中2問不正解なら
  • 7月法令基礎記事の主体表と往復·7月第2週10問解き直し3問正解
  • を目標にしてください

3法令10問での検査出題イメージ

2級では手続·主体選択が中心です。

| 出題型 | 例 |
| --- | --- |
| 手続選択 | 性能検査 |
| 主体 | 届出義務者 |
| 再検査 | 不合格後 |
| 誤答 | 検査種類混同 |

  • たとえば9月13:30·40問模試で法令38点·検査2問不正解なら
  • 10月4日まで検査10問·10月11日12:45到着·本番40問
  • という直前対策が定番

6よくある質問

検査証と性能検査は暗記量が多いですか？
4行表+主体理解が中心です。たとえば6月4行表·法令検査10問·10月11日13:30本番まで月1回見直し、9月法令40点以上2回、各40点足切りも確認、が定番です。期限等の数値は公式テキストの規定どおり確認してください。
法令基礎記事との違いは？
基礎記事は4ブロック·20問測定、本記事は検査証·性能検査の4行比較·主体混同、が中心です。例えば6月基礎·6月第3週本記事、7月検査種類·更新記事、9月240点·各40点以上2回、という順が定番です。
主任者記事との関連は？
検査手続と主任者選任は別論点ですが同一10問内に並ぶことがあります。たとえば6月本記事検査4行·7月主任者10問、9月40問模試法令42点以上、主体混同に注意、10月11日13:30本番まで別タグで記録してください。
