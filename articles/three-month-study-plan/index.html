3か月で2級ボイラー技士を目指す学習計画の立て方

2級ボイラー技士試験（40問·400点·180分）の3か月計画は、残り週数カウントから5ステップで表に落とすのが定番です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、7月開始·13週·週6時間78時間·Step1資格確認·Step2残り週·Step3週次表·Step4模試2回·Step5申込8,800円、から計画表を作ります。

この記事の要点

本記事を読むと、たとえば7月開始·13週·5ステップの計画表作成手順が分かります。

  • 5ステップ
  • 13週逆算
  • 78時間
  • 模試2回
  • 8,800円申込

15ステップで計画表を作る（資格→週数→時間）

計画立ては5ステップ固定です。

| Step | 内容 |
| --- | --- |
| 1 | 資格·40問·240点 |
| 2 | 残り週数 |
| 3 | 週6h×13＝78h |
| 4 | 模試2回 |
| 5 | 8,800円申込 |

たとえば6月6日Step1公式メモ·7月1日開始13週·10月11日13:30本番まで·9月模試240点·各40点以上2回、10月11日13:30本番まで、という表1枚が定番です。

213週以内かの判定（18週以上なら6ヶ月計画へ）

残り週数で計画記事を選びます。

| 残り | 記事 |
| --- | --- |
| 13週 | 3か月本記事 |
| 18週 | 6ヶ月計画 |
| 26週 | 6ヶ月週次表 |
| 52週 | 1年計画 |

  • 例えば6月6日残り18週·7月開始なら13週で可·6月開始なら18週で6ヶ月計画
  • 8月8

  • 9月13:30模試240点·各40点
  • 10月11日13:30本番まで
  • を目標にしてください

3週次表テンプレ（弱点3h·維持2h·用語1h）

78時間を週6時間に配分します。

| ブロック | 時間 |
| --- | --- |
| 弱点 | 3時間 |
| 維持 | 2時間 |
| 用語 | 1時間 |
| 計 | 6時間/週 |

  • 第11週13:30模試240点·各40点以上
  • 10月11日12:45到着·本番
  • 400点満点·180分·40問が本番条件
  • という流れが定着しやすい

43ヶ月学習計画記事との使い分け

立て方と週次表は別記事です。

| 記事 | 内容 |
| --- | --- |
| 本記事 | 5ステップ·表作成 |
| 3ヶ月計画 | 13週週次詳細 |
| 再受験 | 前回得点表 |

  • 例えば6月本記事5ステップ·6月第2週3ヶ月計画週次·9月模試240点·各40点以上2回
  • 10月11日13:30本番まで
  • という順が定番

513週目までの計画立てロードマップ

10月11日（日）13:30本番までの例です。

1. 第0週：5ステップ表作成
2. 第1週：40問通し·分析
3. 第9週：8,800円申込
4. 第11週：13:30·模試
5. 第13週：12:45到着·本番

  • たとえば第11週模試240点·各40点以上なら計画成功
  • 法令38点なら第12週まで法令10問/日追加·400点満点·180分
  • という仕上げが定番

6よくある質問

3か月計画はいつから始められますか？
本番13週前が目安です。たとえば10月11日13:30本番·7月1日開始13週·Step1資格確認完了、残り18週ある6月6日なら6ヶ月計画も選択可。9月模試240点·各40点以上2回、10月11日13:30本番まで週6時間維持が定番です。
3ヶ月学習計画記事との違いは？
本記事は計画表作成5ステップ、3ヶ月計画記事は13週週次配分詳細、が中心です。例えば6月本記事5ステップ·6月第2週3ヶ月計画週次、9月40問模試240点·各40点以上2回、10月11日13:30本番まで両方参照が定番です。
基礎未完了でも3か月計画は可ですか？
再受験者·基礎済み向けです。たとえば第1週40問200点未満·残り18週なら6ヶ月計画へ、8,800円申込前に資格·40問·240点確認、400点満点·180分·40問が本番条件です。ボイラー検定（公式）で受験資格を確認してください。
