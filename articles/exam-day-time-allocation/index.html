2級ボイラー技士試験の試験当日の時間配分

2級ボイラー技士試験は13:30開始·約3時間·40問（4科目×各10問）のマークシートです。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、第16週の日曜に13:30から40問演習を1回解き、約4.5分/問·見直し15分の配分を体験しておくのが定番です。正本はボイラー検定（公式）の最新要項です。

この記事の要点

本記事を読むと、たとえば科目ごと45分×4＋見直し15分の時間配分例が表で整理され、40問·400点·各科目40点足切りを意識した本番運用ができます。

  • 13:30〜16:30
  • 約4.5分/問
  • 4科目×45分
  • 見直し15分
  • 足切り40点意識

140問·180分の基本配分

2級ボイラー学科試験は、おおむね13:30〜16:30（約3時間）で40問に解答します。1問あたり約4.5分が目安です。

| 項目 | 内容 |
| --- | --- |
| 問題数 | 40問（4科目×各10問） |
| 時間 | 約180分（3時間） |
| 1問目安 | 約4.5分 |
| 見直し | 10〜15分確保 |

  • たとえば13:30開始なら
  • 14:15頃に第1〜10問（構造）を終え
  • 15:00頃に第20問まで進むペースが目安

50問は他資格の数値（2級は40問）と混同しないでください。

24科目×10問の科目別配分例

出題は4科目（構造·取扱い·燃料燃焼·関係法令）×各10問です。均等配分が基本ですが、苦手科目に時間を寄せる調整も有効です。

| 科目 | 問題番号 | 配分目安 |
| --- | --- | --- |
| 構造 | 第1〜10問 | 45分 |
| 取扱い | 第11〜20問 | 45分 |
| 燃料燃焼 | 第21〜30問 | 45分 |
| 関係法令 | 第31〜40問 | 45分 |

例えば関係法令が苦手なら第31〜40問に55分·構造を35分に短縮するなど、演習で正答率40％未満の科目に+10分を配分します。

3見直し15分の使い方

最終10〜15分は、空欄·マーク漏れ·消し跡の確認に充てます。各科目40点の足切りを意識し、苦手科目を優先して見直します。

| 優先度 | 確認内容 |
| --- | --- |
| 1 | 未マーク·空欄 |
| 2 | 消しゴムカス·薄いマーク |
| 3 | 足切り40点未満の科目 |
| 4 | 迷った2択の最終判断 |

たとえば16:15頃に40問すべてにマークがあるか確認し、16:25のアナウンス後は消しゴムでカスを落とさないよう注意します。総合280点でも1科目38点なら不合格です。

4時間が足りなくなったときの判断

残り10問で15分しかない場合、1問あたり1.5分以下になります。完全に解けない問題は、消去法でマークして次へ進みます。

| 残り時間 | 行動 |
| --- | --- |
| 30分·10問 | 3分/問で解く |
| 15分·10問 | 消去法中心 |
| 5分·5問 | 空欄ゼロを優先 |
| 最終1分 | 未マークを埋める |

  • 例えば第36問で3分悩んだら
  • 残り5問に12分しかないため
  • 第37〜40問は1問2分で消去法マークに切り替え

空欄のまま提出しないことが各科目40点確保の最低条件です。

5直前1週間の時間配分リハーサル

本番前に、13:30開始·3時間の40問演習で配分を体験します。

1. 13:30開始·40問·タイマー180分をセット
2. 科目ごと45分·見直し15分で進める
3. 45分経過時点の到達問数を記録
4. 180分で40問すべてにマークできたか確認
5. 足りない科目に翌週+2時間を追加

  • たとえば10月4日（日）13:30から40問演習を解き
  • 16:30頃に終了·見直し15分を確保できたか振り返ると
  • 10月11日（日）本番の時間感覚が身につき

6よくある質問

1問に何分かければよいですか？
40問·約180分なら、おおむね4.5分/問が目安です。たとえば13:30開始なら14:15頃に10問目、15:00頃に20問目まで進むペースを意識してください。計算問題で5分超えそうなら、消去法でマークして次の問題へ進み、見直し15分で再挑戦する運用が現実的です。
科目の順番は固定ですか？
問題用紙の並びは、構造·取扱い·燃料燃焼·関係法令の4科目×各10問が一般的です。例えば第1〜10問が構造·第31〜40問が関係法令なら、苦手な法令に55分·得意な構造を35分と配分を調整できます。ただし問題番号順に解く方が迷いが少ないため、初受験者は順番どおり進めることをおすすめします。
見直し時間はどのくらい確保すべきですか？
10〜15分確保する受験者が多いです。たとえば16:15から見直しを開始し、40問すべてにマークがあるか·空欄がないかを確認します。各科目40点の足切りを意識し、演習で正答率40％未満だった科目を優先して見直してください。直前1週間の40問演習で見直し15分を実際に使ってみると定着します。
