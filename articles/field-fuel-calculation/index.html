1燃料10問中の計算配分（2〜4問）

燃料科目10問中、計算·数値はおおむね2〜4問です。

| タイプ | 例 |
| --- | --- |
| 空気比 | 過多·不足·％ |
| 発熱量 | kcal/h換算 |
| 排ガス | 温度·成分 |
| 理論のみ | 6〜8問 |

  • たとえば6月第1週燃料10問·計算3問中1問不正解なら
  • 空気比論点に週2時間を集中
  • 10月11日13:30本番まで残り18週
  • と配分を決め

2単位換算表（kW·kcal/h·％）

計算ミスの多くは単位不一致です。

| 単位 | 用途 |
| --- | --- |
| kW | 出力 |
| kcal/h | 熱量 |
| ％ | 空気比·効率 |
| t/h | 蒸気量 |

  • 例えば6月第2週換算表写し10分·燃料計算5問·解説20分
  • 火·金に1時間固定·8月8
  • 800円申込前まで12回
  • というセットが定着しやすい

50問試験等の他資格情報と混同しないでください。

| ステップ | 内容 |

| 1 | 求める量·単位を下線 |
| 2 | 既知量の単位を揃える |
| 3 | 式·桁·選択肢照合 |
| 4 | 1問4〜5分で次へ |

6よくある質問

燃料の計算は電卓で解けますか？
2級ではマークシート·電卓不可が一般的です。たとえば6月から筆算·概算·10月11日13:30本番まで週2時間·計算演習20問×9回、で1問5分以内を目指してください。単位換算表を演習前に10分見直す、が定番の習慣です。
燃料基礎記事との違いは？
基礎記事は4ブロック·20問測定·章立て把握、本記事は計算·数値2〜4問の単位·式·誤答分類が中心です。例えば6月基礎記事·6月第3週本記事計算10問、7月科目対策記事、9月燃料40点以上2回、という順が定番です。
計算が苦手でも燃料40点は取れますか？
理論6〜8問の得点で40点は可能です。たとえば10問中理論7問正解·計算3問全不正解でも70点、ただし計算2問正解追加で足切り余裕が増えます。残り18週·週2時間計算、9月模試燃料42点以上、を目標にしてください。
