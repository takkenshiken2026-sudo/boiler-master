2級ボイラー技士試験の関係法令の過去問での出題傾向

2級ボイラー技士試験の関係法令科目（各10問）の出題傾向は、演習40問（10問×4セット）のタグ分析と9月40問模試2回で把握するのが定番です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月·7月に法令10問×4セットを届出·検査·主任者·区分でタグ分析し、出現率上位2論点に8月まで週3時間を集中します。

この記事の要点

本記事を読むと、たとえば法令40問のタグ分析表と出現率上位2論点への集中配分が分かります。

  • 40問タグ分析
  • 出現率上位2
  • 模試2回
  • 法令40点
  • 18週逆算

| 検査 | 6〜10問 |
| 主任者 | 6〜10問 |
| 区分 | 4〜8問 |
| 主体 | 4〜8問 |

  • 計40問タグ付け·10月11日13:30本番まで残り18週·出現率上位2特定

2出現率上位2論点への集中配分

全論点均等より、上位2論点を厚くします。

| 順位 | 配分例 |
| --- | --- |
| 1位 | 週2h·演習10問 |
| 2位 | 週1h·演習10問 |
| 3位以下 | 週30分維持 |
| 8月 | 8,800円申込 |

  • 例えば届出14問·主任者10問が上位なら
  • 6〜8月週3時間を届出·主任者
  • 9月模試法令42点以上
  • を目標にしてください

34セット集計の注意点

2級は40問·4科目固定です。

| 注意 | 内容 |
| --- | --- |
| 50問混同 | 2級は40問 |
| 1回だけ | 最低4セット |
| 足切り | 各40点 |
| 条文暗記 | 骨格理解 |

  • たとえば6月セット1で届出5問
  • 7月セット2で4問
  • 合計40問で届出12問
  • という集計が安定し

280点·法令38点は各40点足切りで不合格です。

6よくある質問

法令40問で傾向分析は足りますか？
10問×4セット+解き直し20問、9月40問模試2回、が定番の量です。たとえば6〜8月法令40問·10月11日13:30本番、で十分な傾向分析が可能、と考えてください。量よりタグ分析の質を優先します。
現場経験で傾向分析は省略できますか？
演習得点で判断してください。たとえば現場10年·法令55点でも届出論点2問不正解なら傾向分析で届出を特定、9月法令42点以上2回、が定番です。280点·法令39点は各40点足切りで現場経験があっても不合格です。
取扱いの出題傾向記事との順序は？
科目別に並行可能です。例えば6月法令40問分析·7月取扱い40問分析、8月8,800円申込·9月40問模試2回240点·各40点以上、4科目得点を別記録し、10月11日13:30本番まで確認してください。
