2級ボイラー技士試験の模試日程と学習計画

2級ボイラー技士試験の独学では、40問·180分·13:30開始の模試を学習計画に組み込むと本番の時間感覚が身につきます。たとえば10月11日（日）13:30本番まで18週の6月6日なら、8週目·12週目·17週目（10月4日）に40問模試を1回ずつ、240点·各40点以上を記録するのが定番です。50問模試は2級（40問）と混同しないでください。

この記事の要点

本記事を読むと、たとえば「8月·9月·10月4日」の3回模試日程と、各回後の弱点科目追加が18週カレンダーに組み込めます。

  • 18週中3回模試
  • 13:30·180分
  • 40問·各40点
  • 10月4日最終
  • 弱点補強

118週中の模試3回配置

残り18週の学習計画に、40問本番形式模試を3回組み込みます。

| 回 | 週 | 目的 |
| --- | --- | --- |
| 1回目 | 8週目 | 初期得点把握 |
| 2回目 | 12週目 | 中間チェック |
| 3回目 | 17週目 | 本番7日前 |

たとえば6月6日開始·10月11日13:30本番なら、8月第1週·9月第1週·10月4日（日）13:30に各180分·40問模試を実施します。各回240点·各40点以上を記録してください。

2模試当日の設定（13:30·180分）

本番に近い条件で受けると効果が高いです。

| 設定 | 内容 |
| --- | --- |
| 開始 | 13:30 |
| 時間 | 180分（見直し15分含む） |
| 問題 | 40問·4科目×10問 |
| 参照 | テキスト見ない |

例えば10月4日（日）12:45着席·13:00鉛筆準備·13:30スタート·科目45分×4·見直し15分、という流れで本番10月11日と同じリズムを体験してください。

31回目模試後（8週目）の調整

1回目で240点未満·40点未満科目があれば、残り10週の配分を変更します。

| 1回目得点 | 残り10週 |
| --- | --- |
| 220点·法令36点 | 法令週3h |
| 260点·全40点+ | 維持·2回目へ |
| 200点未満 | 週6時間に増 |
| 320点以上 | 2回目で確認 |

  • たとえば8月第1週模試240点·関係法令38点なら
  • 8月第2週〜9月第4週で法令演習20問·9月第1週2回目模試
  • とスケジュールし

42回目模試後（12週目）の調整

2回目で320点·全科目40点以上なら、直前は間違いノート中心に切り替えます。

| 2回目 | 残り5週 |
| --- | --- |
| 320点+ | 弱点10問/週 |
| 280点 | 最弱科目週2h |
| 240点- | 基礎+演習量 |
| 足切り1科目 | その科目集中 |

  • 例えば9月第1週模試300点·燃料42点·取扱い38点なら
  • 9月第2週〜10月3日に取扱い演習20問
  • 10月4日3回目模試·10月11日13:30本番
  • という流れが定番

53回目模試（10月4日）以降

本番7日前の模試後は、新規インプットを減らします。

1. 10/4：13:30·40問模試·得点分析
2. 10/5〜9：40点未満科目30分/日
3. 10/10 20:00：持ち物5点·封筒

  • たとえば10月4日模試320点·全科目40点以上なら
  • 10月5日以降は間違いノート10問/日·前日持ち物準備·当日40問本番
  • に切り替えてください

6よくある質問

模試は年間何回受けるべきですか？
18週計画なら3回（8週·12週·17週目）が目安です。たとえば10月11日13:30本番まで18週なら、8月·9月·10月4日に各40問·180分·13:30開始で受け、240点·各40点以上を記録します。直前模試の受け方記事では本番7日前1回に絞る方法も解説しています。
本サイトの演習で模試日程は組めますか？
過去問形式演習40問を選び、カレンダーに13:30·180分を登録すれば模試日程になります。例えば構造10問·取扱い10問·燃料10問·法令10問を8月第1週日曜13:30に通し解き、得点を4科目別に記録してください。時間を計った演習記事も参照できます。
模試で240点未満だったら日程をずらしますか？
試験日変更は要項上難しいため、残り週数で弱点科目の時間を増やす方が現実的です。たとえば8週目模試220点·残り10週なら、週5時間×10＝50時間追加で40点未満科目の演習20問を集中させ、12週目2回目模試で260点を目標にします。不合格時の日程調整は再受験記事を参照してください。
