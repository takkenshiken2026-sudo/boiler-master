ボイラー事故から学ぶ安全論点｜2級試験に活かす読み方

2級ボイラー技士試験（40問·400点·180分）では、事故事例から逆算した安全論点が取扱い·関連法令科目で問われます。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週低水位·過圧·スケール3事故比較表·取扱い演習10問·9月模試240点·各40点以上2回、という事故読み方から始めます。

この記事の要点

本記事を読むと、たとえば6月第1週の3事故比較表·取扱い10問と9月240点·各40点確認が分かります。

  • 3事故比較
  • 低水位·過圧
  • 安全装置
  • 取扱い10問
  • 9月模試2回

1試験で押さえる3事故（低水位·過圧·スケール）

事故は3類型で整理します。

| 事故 | 原因 |
| --- | --- |
| 低水位 | 水位管理不全 |
| 過圧 | 安全弁·圧力管理 |
| スケール | 水質·過熱 |
| 共通 | 点検·装置 |

たとえば6月第1週演習10問·低水位関連2問·10月11日13:30本番まで残り18週·7月安全装置記事とセット、9月模試240点·各40点以上2回、400点満点·180分·取扱い40点足切りが最優先です。

2事故から逆算する安全装置（低水位·燃料遮断）

装置は事故防止の答えです。

| 装置 | 防止 |
| --- | --- |
| 低水位警報 | 干ばく |
| 燃料遮断 | 過熱 |
| 安全弁 | 過圧 |
| 連続ブロー | スケール |

  • 例えば6月第2週10問·低水位警報→燃料遮断1問·安全弁作動条件1問
  • 9月13:30模試240点·各40点以上2回
  • 10月11日13:30本番まで
  • 安全装置一覧記事とセット復習が定番

3演習10問での事故タグ付け

誤答は2分類で記録します。

| 分類 | 例 |
| --- | --- |
| 装置未作動 | 警報無視 |
| 操作ミス | ブロー不足 |
| 水質 | 硬度·pH |
| 燃焼 | 空気比 |

  • たとえば6月10問·装置未作動3問·操作2問
  • 7月比較表1枚追加·9月14日模試240点·各40点以上
  • 10月11日12:45到着·本番
  • 400点満点·180分·という改善が定着しやすい

6よくある質問

事故事例を全部覚える必要がありますか？
3類型の原因·対策·装置で足ります。たとえば6月低水位·過圧·スケール比較表1枚·取扱い10問で確認、9月模試240点·各40点以上2回、10月11日13:30本番まで装置と原因の組み合わせが定番です。400点満点·180分·40問が本番条件です。
安全装置記事との違いは？
本記事は事故から逆算·3類型、安全装置記事は装置一覧·作動条件、が中心です。例えば6月本記事3事故表·6月第2週安全装置記事、9月40問模試240点·各40点以上2回、10月11日13:30本番まで両方参照が定番です。
関連法令科目でも出ますか？
安全規制·検査関連で出題あり。たとえば6月関連10問中安全規制2問·取扱い10問中事故関連2問、9月模試240点·各40点以上2回、各40点足切り最優先、400点満点·180分が本番条件です。10月11日13:30本番まで関連10問復習が定番です。
