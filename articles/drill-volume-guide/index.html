2級ボイラー技士試験の演習問題量の目安

2級ボイラー技士試験（40問·400点·240点合格·各40点足切り）の演習量は、「何問解けば足りるか」より「240点·各40点を2回連続で確認できるか」で判断するのが定番です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、40問×5回＝200問+解き直し50問·週5時間×18週＝90時間、を目安にカレンダーへ書き込みます。

この記事の要点

本記事を読むと、たとえば200問+解き直し50問·9月40問模試2回の演習量目安と正答率別の追加量が分かります。

  • 200問+解き直し
  • 90時間目安
  • 240点2回確認
  • 各40点記録
  • 9月模試2回

1演習量の基本目安（200問+解き直し50問）

2級独学の現実的目安は次のとおりです。

| 項目 | 目安 |
| --- | --- |
| 新規演習 | 200問（40問×5回） |
| 解き直し | 50問（25%） |
| 学習時間 | 80〜90時間 |
| 合格確認 | 240点·各40点×2回 |

  • たとえば6月6日·10月11日13:30試験·残り18週·週5時間＝90時間なら
  • 6月40問·7月40問·8月40問·9月模試2回·10月解き直し
  • という配分が定番

2正答率別の追加演習量

初回40問の正答率で、追加量を決めます。

| 正答率 | 追加目安 |
| --- | --- |
| 70％+ | 200問維持 |
| 60〜69％ | +40問 |
| 50〜59％ | +80問·週6h |
| 50％未満 | +120問·要因分析 |

  • 例えば6月第1週40問220点（55％）·法令38点なら
  • 7月週6時間·演習80問·9月240点·各40点以上
  • 2週連続確認
  • を目標にしてください

3科目別10問の最低ライン

40問演習では、4科目×10問の得点を必ず記録します。

| 科目 | 足切り | 目標 |
| --- | --- | --- |
| 構造 | 40点 | 45点+ |
| 取扱い | 40点 | 45点+ |
| 燃料 | 40点 | 45点+ |
| 法令 | 40点 | 45点+ |

  • たとえば8月40問280点·燃料39点なら
  • 9月まで燃料演習30問追加·10月11日13:30本番前に燃料42点以上2回
  • という調整が定番

4解き直し50問の配分

解き直しは、弱点科目を厚くします。

| 科目 | 解き直し目安 |
| --- | --- |
| 40点未満 | 20問 |
| 40〜49点 | 10問 |
| 50点以上 | 5問 |
| 計 | 50問前後 |

  • 例えば6月220点·法令38点·燃料48点なら
  • 解き直し20問を法令
  • 10問を燃料
  • 残り20問を全体
  • 3日·7日サイクルで9月まで消化
  • という配分が現実的

5残り18週の演習カレンダー例

10月11日（日）13:30本番までの例です。

1. 6月：40問×1·得点分析
2. 7月：40問×1·弱点30問
3. 8月：40問×1·8,800円申込
4. 9/7·9/14：13:30模試×2
5. 10/11：12:45到着·本番40問

  • たとえば9月14日模試240点·各40点以上なら演習量十分
  • 235点なら9月第3週解き直し20問追加
  • という判断が定番

6よくある質問

最低何問解けば合格できますか？
問題数より240点·各40点の再現性が重要です。たとえば200問+解き直し50問·9月40問模試2回で240点·各40点以上、が定番の目安です。50問·100問試験等の他資格情報と混同しないでください。2級は常に40問·400点満点です。
演習量と学習時間の関係は？
1問あたり4〜5分換算が目安です。例えば200問×4分＝800分（約13時間）は演習のみ·解説·用語確認含むと80〜90時間になります。残り18週·週5時間＝90時間なら200問+解き直し50問が現実的、週3時間なら120問+解き直し30問、または試験回延期を検討してください。
時間を計った演習記事との違いは？
本記事は総問題数·解き直し比率·90時間の逆算が中心です。たとえば「200問+50問·9月模試2回」の量設計向けです。時間を計った演習記事は13:30·180分·4.5分/問の本番ペース確認向け、過去問活用法記事は解きっぱなし防止の手順向けです。
