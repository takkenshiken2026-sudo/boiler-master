2級ボイラー技士試験の不合格からの再受験計画

2級ボイラー技士試験不合格後は、前回40問得点表から40点未満科目を特定し、次回8,800円再申込まで18週で弱点を潰します。たとえば11月合格発表·前回280点·法令38点なら、12月法令週3時間·翌春8,800円申込·9月模試240点·各40点以上2回、という再受験計画から始めます。

この記事の要点

本記事を読むと、たとえば11月280点·法令38点·18週で法令週3時間·10問/週の再受験計画と翌春240点·各40点確認が分かります。

  • 前回得点表
  • 40点未満優先
  • 18週逆算
  • 8,800円再申込
  • 配分変更

1前回得点表と足切り判定（280点·38点の典型）

不合格は2パターン。

| パターン | 例 |
| --- | --- |
| 足切り | 280点·38点 |
| 総合不足 | 220点·全42点 |
| 合格 | 240点·各40点 |
| 満点 | 400点 |

  • たとえば11月280点·法令38点·足切り不合格
  • 12月得点表作成·9月模試240点·各40点以上2回
  • 400点満点·180分·8
  • 800円再申込が必要

240点未満科目への週3時間追加配分

弱点は時間追加。

| 科目 | 追加 |
| --- | --- |
| 40点未満 | 週3時間 |
| 40-44点 | 週1時間 |
| 45点以上 | 維持 |
| 演習 | 10問/週 |

  • 例えば法令38点·週3時間·10問/週·18週54時間
  • 9月13:30模試240点·各40点以上2回
  • 10月11日13:30本番まで
  • 点数分析記事とセット確認が定番

3前回と異なる勉強法への変更

同じ方法は再失敗。

| 変更 | 例 |
| --- | --- |
| 配分 | 法令+3時間 |
| 教材 | 新版確認 |
| サイクル | 3日7日 |
| 模試 | 2回必須 |

  • たとえば前回40問/週·法令38点再発·変更後10問/日·42点
  • 9月14日模試240点·各40点以上
  • 400点満点·180分·再受験対策記事とセットが定番

8,800円再申込前に弱点科目を確認してください。

4再受験対策·日程調整記事との接続

再受験は記事セットです。

| 記事 | 内容 |
| --- | --- |
| 本記事 | 得点表起点 |
| 再受験対策 | 弱点潰し |
| 日程調整 | 申込日 |
| 点数分析 | 判定表 |

例えば11月本記事得点表·12月再受験対策·翌春8,800円申込·9月模試240点·各40点以上2回、ボイラー検定（公式）で次回日程確認が定番です。400点満点·180分·40問が本番条件です。

6よくある質問

前回と同じ勉強法で大丈夫ですか？
同じ方法では足切り残存。たとえば前回260点·法令38点·同40問/週、再び38点リスク、6月法令週3時間·10問/週変更、9月模試240点·各40点以上2回、400点満点·180分·40問が本番条件です。
3ヶ月計画記事との違いは？
3ヶ月計画は初受験向け、本記事は再受験·前回得点表起点、が中心です。例えば前回38点科目ありなら本記事·初受験なら3ヶ月計画、9月40問模試240点·各40点以上2回、10月11日13:30本番まで使い分けが定番です。
再受験でも新教材は必要ですか？
版·要項一致が優先。たとえば6月版確認·7月法令改正章更新·8,800円申込前一致、前回解きっぱなし3日7日解き直し、9月模試240点·各40点以上2回、400点満点·180分·各40点足切り最優先、10月11日13:30本番まで新版テキスト確認してください。
