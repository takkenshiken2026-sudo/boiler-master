似た選択肢の見分け方｜2級ボイラー技士のひっかけ対策

2級ボイラー技士試験（40問·180分）では、似た4択の一字違い·数値違い·否定·部分正解がひっかけの典型です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、6月第1週5陷阱表·消去法10問·9月模試240点·各40点以上2回、という似た選択肢対策から始めます。

この記事の要点

本記事を読むと、たとえば6月第1週の5陷阱表·消去法10問と9月240点·各40点確認が分かります。

1似た選択肢5陷阱（一字·数値·順序·否定·部分正解）

陷阱は5種類表。

| 陷阱 | 例 |
| --- | --- |
| 一字 | 遮断/遮止 |
| 数値 | 桁違い |
| 順序 | 前後逆 |
| 否定 | ない/する |
| 部分 | 半分正解 |

  • たとえば6月第1週5陷阱表·演習10問·否定2問
  • 10月11日13:30本番まで残り18週·7月消去法練習
  • 9月模試240点·各40点以上2回
  • 400点満点·180分·4.5分/問目安維持

2消去法10問（明確誤り2肢除外→2肢比較）

消去法は3ステップ。

| 順 | 内容 |
| --- | --- |
| 1 | 2肢除外 |
| 2 | 残り2肢 |
| 3 | 1行差 |
| 演習 | 10問 |

  • 例えば6月10問·2肢除外8問·正解6問
  • 9月13:30模試240点·各40点以上2回
  • 10月11日13:30本番まで
  • 混同用語記事とセット確認が定番

400点満点·180分·各40点足切り最優先。

  • 400点満点·180分·数値引っかけ記事参照

6よくある質問

混同用語記事との違いは？
本記事は4択陷阱·消去法、混同記事は用語5ペア比較、が中心です。例えば6月本記事5陷阱·6月第2週混同5ペア、9月40問模試240点·各40点以上2回、10月11日13:30本番まで両方参照が定番です。
消去法は時間がかかりませんか？
10問練習で定着。たとえば6月10問·2肢除外·4.5分/問、9月模試240点·各40点以上2回、400点満点·180分·40問·8,800円申込が本番条件です。10月11日13:30本番まで部分正解陷阱に特に有効です。
数値引っかけ記事との違いは？
本記事は5陷阱全体·消去法、数値記事は桁·単位·逆算、が中心です。例えば6月本記事5陷阱·6月第2週数値、9月模試240点·各40点以上2回、400点満点·180分·各40点足切り最優先、10月11日13:30本番まで両方参照してください。
