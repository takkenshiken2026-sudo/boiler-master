ボイラー区分の覚え方｜小型・簡易・伝熱面積

2級ボイラー技士試験の関係法令（各10問）では、小型ボイラー·簡易ボイラー·一般ボイラーの区分と伝熱面積·蒸気量などの基準が頻出です。たとえば10月11日（日）13:30開始まで残り18週の6月6日なら、公式テキストの区分表を1枚にまとめ、法令演習10問で「伝熱面積だけ見れば足りる」誤答肢を除外する練習から始めるのが定番です。数値基準は公式テキストが正本です。

この記事の要点

本記事を読むと、たとえば小型·簡易·一般の比較表と法令10問中1〜2問の区分問題の解き方が分かります。

  • 伝熱面積が軸
  • 3区分比較表
  • 法令10問
  • 誤答肢に注意
  • テキスト正本

1区分を覚える3軸（伝熱面積·蒸気量·用途）

試験では、区分の「判断軸」をセットで問われます。

| 軸 | 試験での役割 |
| --- | --- |
| 伝熱面積 | 小型·一般の境界 |
| 蒸気量·出力 | 簡易等の条件 |
| 用途·設置 | 特例·届出 |
| 正本 | 公式テキスト区分表 |

  • たとえば6月第1週に公式テキストの区分表を写し
  • 10月11日13:30本番まで残り18週·法令10問中2問が区分系
  • と想定して週1時間を区分暗記に配分し

数値は年度で改正されるためテキストで確認してください。

2小型·簡易·一般の比較表の作り方

暗記より、自分用の比較表を1枚作るのが定番です。

| 区分 | 覚える項目 |
| --- | --- |
| 小型 | 伝熱面積上限·検査 |
| 簡易 | 蒸気量·出力·特例 |
| 一般 | それ以外·手続 |
| 混同 | 1級·3級の区分 |

  • 例えば6月第2週の法令演習10問で
  • 「小型＝蒸気量だけで判定」型の誤答肢が2問出たら
  • 比較表の「判断軸」列を追記してください

40問·各科目40点足切りでは法令38点でも不合格になります。

3法令10問での出題パターン

2級では詳細な計算より、区分·届出·検査の組み合わせが中心です。

| 出題型 | 例 |
| --- | --- |
| 区分選択 | 伝熱面積○m²の帰属 |
| 手続 | 設置届の要否 |
| 誤答排除 | 単一条件だけで判定 |
| 主体 | 届出者·検査 |

  • たとえば10月4日40問模試で法令9問正解·区分1問不正解なら
  • 10月11日13:30本番まで毎日5分·区分表を見直す
  • という直前対策が定番

関係法令は4科目のうち10問です。

4よくある混同（1級·衛生管理者·50問）

他資格·上級区分と混同しないでください。

| 混同 | 正しい整理 |
| --- | --- |
| 1級の区分 | 2級テキストの表 |
| 50問·100問 | 2級は40問 |
| 伝熱面積のみ | 蒸気量等も確認 |
| 口コミ数値 | テキスト正本 |

  • 例えば6月第3週
  • 法令演習20問中区分3問·届出2問·作業主任者2問
  • とタグ付けし
  • 区分3問だけ比較表と往復すると効率が上がり

240点合格·各40点以上を維持してください。

5区分演習10問の1週間サイクル

区分は、短周期で解き直すと定着します。

1. 月：テキスト区分表を写し
2. 火：法令演習10問（区分中心）
3. 木：誤答3問を用語解説で確認
4. 土：同じ10問を解き直し
5. 日：8問以上正解で次章へ

たとえば6月第2週10問→6月第3週解き直しで7問→10問正解なら定着、6問以下なら設置届記事とセットで手続の流れも1枚にまとめてください。8,800円申込前に40問·240点の全体計画も確認します。

6よくある質問

伝熱面積の数値は暗記必須ですか？
公式テキストの区分表に載る数値は法令10問で問われ得ます。たとえば6月第1週にテキスト該当ページを写し、10月11日13:30本番まで週30分·計9時間で見直す、というサイクルが定番です。数値は改正で変わるため、ボイラー検定（公式）·テキスト最新版で必ず確認してください。
小型と簡易ボイラーの違いは何ですか？
判断軸（伝熱面積·蒸気量·出力等）が異なります。例えば法令演習10問で「簡易＝小型の別名」型の誤答肢に注意し、自分用比較表の3行（小型·簡易·一般）に判断条件を書き込んでください。2級は40問·各科目40点足切りです。区分だけ覚えて手続を落とさないよう、設置届記事も合わせて読むとよいです。
区分だけで受験できますか？
いいえ。関係法令10問は届出·検査·作業主任者等も含みます。たとえば残り18週·週5時間なら、区分2時間·届出2時間·主任者1時間、と配分し、9月40問模試で法令40点以上を2回連続確認してください。280点·法令38点は不合格の典型例です。
