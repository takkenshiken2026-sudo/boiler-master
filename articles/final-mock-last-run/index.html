2級ボイラー技士試験の直前模試の受け方

2級ボイラー技士試験の直前模試は、13:30開始·約3時間·40問（4科目×各10問）の本番形式で1回受けるのが定番です。たとえば10月11日（日）本番まで1週間の10月4日（日）なら、13:30から180分タイマーで40問演習を解き、240点·各科目40点以上を満たしているか分析します。正本の合格基準はボイラー検定（公式）の要項です。

この記事の要点

本記事を読むと、たとえば試験1週間前の40問模試と320点以上·40点未満科目の追加対策が分かり、本番の時間配分（約4.5分/問）を体験できます。

  • 1週間前·40問
  • 13:30·180分
  • 各40点足切り
  • 弱点科目特定
  • 前日は模試しない

1直前模試は試験1週間前·40問1回

直前模試の最適タイミングは、本番約1週間前です。13:30開始·180分·40問を1セットで受けます。

項目内容
タイミング本番7日前（例10月4日）
開始13:30（本番同様）
時間180分（約3時間）
問題数40問（4科目×10問）

たとえば10月11日（日）13:30本番なら、10月4日（日）13:30から40問模試を解き、16:30頃終了·見直し15分を確保できたか振り返ります。50問は他資格（2級は40問）と混同しないでください。

2模試当日の環境設定

本番に近い環境で受けると、時間感覚が身につきます。

設定内容
タイマー180分·13:30開始
用具HB鉛筆·消しゴム·マークシート
中断原則なし（トイレは開始前）
参照テキスト見ない

例えば10月4日（日）12:45に席につき、13:00にHB鉛筆2本を準備·13:30スタート、科目ごと45分·見直し15分の配分で進めます。スマホは別室に置き、会場時計の代わりにタイマーを前方に置いてください。

3模試後の得点分析（240点·各40点）

模試終了後は、総合点と科目別得点を記録します。合格基準は240点以上·各科目40点以上です。

得点判断
320点以上·全科目40点+本番合格圏内
240〜319点弱点科目に集中
1科目40点未満最優先対策
240点未満基礎＋演習量追加

たとえば模試280点·関係法令38点なら、10月5〜10日に法令演習20問·用語10語を1日30分追加します。総合280点でも1科目38点なら不合格です。

4模試後3日以内の弱点補強

模試で判明した弱点は、本番前に3日以内に手当てします。

弱点対策（例）
構造40点未満装置名称·演習10問
取扱い40点未満水質用語·演習10問
燃料燃焼40点未満計算復習·演習10問
法令40点未満条文数値·演習10問

例えば10月4日模試·燃料燃焼42点·取扱い38点なら、10月5〜7日に取扱い演習20問·10月8〜10日に軽い見直し、10月11日13:30本番、という流れが定番です。

5模試後〜本番前日までの過ごし方

模試後は新規インプットを減らし、間違いノート中心に切り替えます。

1. 模試翌日：誤答40問の理由をノートに追記
2. 残り6日：40点未満科目に1日30分
3. 残り3日：用語10語·演習10問/日
4. 前日：模試しない·持ち物準備
5. 当日：12:45到着·40問本番

たとえば10月4日模試·10月10日前日·10月11日（日）12:45会場到着、というカレンダーを書いておくと直前1週間がスムーズです。

6よくある質問

直前模試は何回受けるべきですか？
本番1週間前の40問·180分·13:30開始の本番形式1回が目安です。たとえば10月4日（日）に1回解き、得点分析後に弱点科目へ集中します。前日·前々日に40問通しを重ねると疲労で本番10月11日13:30の集中力が落ちるおそれがあるため、2回目以降は科目別10問程度に留めてください。
模試で240点未満だったらどうすればよいですか？
40点未満科目を特定し、残り1週間でその科目に演習20問·用語10語を集中させます。例えば模試220点·関係法令35点なら、法令演習20問を10月5〜8日に解き、10月9〜10日は間違いノートのみ見直します。新しい参考書は増やさず、240点·各40点の合格基準を要項で再確認してください。
本サイトの演習で模試はできますか？
過去問形式演習を40問選び、13:30開始·180分タイマーで通し解きすれば直前模試の代用になります。たとえば構造10問·取扱い10問·燃料燃焼10問·法令10問をセットし、約4.5分/問·見直し15分で進めてください。模試の活用法·時間を計った演習記事も合わせて参照してください。
