2級ボイラー技士試験の関係法令の事例・ケース問題

1事例問題の4パターン（届出·主任者·検査·休止）

事例型は、状況描写から正しい届出·主体を選びます。

| 状況 | 典型判断 |
| --- | --- |
| 新設 | 設置届 |
| 能力変更 | 変更届 |
| 主任者欠 | 選任義務 |
| 検査遅延 | 再検査 |

  • たとえば6月第1週4事例表を写し
  • 届出·主任者の試験論点を整理
  • 10月11日13:30本番まで残り18週·法令10問中2〜3問が事例型
  • と想定して週2時間配分し

2解き方：状況→主体→届出の3ステップ

文章問題は、読む順序を固定すると安定します。

| ステップ | やること |
| --- | --- |
| 1 | 状況キーワードに線 |
| 2 | 4事例表で届出候補 |
| 3 | 主体（事業者等）確認 |
| 4 | 逆主体の誤答肢排除 |

  • 例えば6月第2週法令10問·事例3問中2問正解なら
  • 7月設置届·主任者記事と往復·7月第2週10問解き直し3問正解
  • を目標にしてください

6よくある質問

現場経験があれば事例は楽ですか？
演習得点で判断してください。たとえば6月10問·事例3問正解なら週1時間維持、2問以下なら週3時間·4事例表復習、9月法令40点以上2回、が定番です。試験の正式届出·制度と現場慣行は異なる場合があります。
設置届記事との関連は？
届出の事例は設置届記事の論点とセットで出得ます。例えば6月本記事4事例·7月設置届10問·解き直し5問、9月40問模試法令42点以上、設置·変更·休止の混同に注意、各40点足切りも必ず確認してください。
法令頻出論点記事との違いは？
頻出論点記事は論点リスト、本記事は事例型の状況→判断·4パターン表、が中心です。たとえば6月基礎·7月頻出·8月本記事事例10問·解き直し5問、9月240点·各40点以上2回、10月11日13:30本番まで、という順が定番です。
